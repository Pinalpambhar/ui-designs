# React & HTML UI Library

A modern collection of **ready-to-use UI components, sections, and full templates** for building fast, beautiful web applications.

This repository helps developers quickly build websites and SaaS products using reusable design blocks in **React** and **HTML**.

---

## ✨ Features

• Reusable UI Components  
• Prebuilt Page Sections  
• Ready-made Website Templates  
• Fully Responsive & Mobile-First  
• Clean and Modern Design  
• Easy Customization  
• Developer Friendly Structure

## ⚛️ React Version

Built with:

- React  
- Vite  
- Tailwind CSS  

### Usage

1. Copy the required component or section  
2. Paste into your project  
3. Adjust props and styles as needed  

---

## HTML Version

Built with:

- HTML5  
- Tailwind CSS  

### Usage

1. Copy the HTML block  
2. Paste into your project  
3. Customize content and styles

## Getting Started

Clone the repository:

git clone https://github.com/pinalpambhar/ui-designs.git

Install dependencies (React version):

npm install  
npm run dev  

---

## Customization

You can easily:

• Change colors via Tailwind config  
• Modify layout spacing  
• Combine sections to create new pages  
• Enable dark mode  
• Extend components with props
